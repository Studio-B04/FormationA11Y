Les points clés

1. Les contrastes

Tous les contenus graphique et les texte de la page doivent être suffisament contrastés par rapport à la couleur de fond afin de permettre une lecture aisée pour les déficients visuels, ou dans des conditions de lumière intense par exemple.

Le contraste, c'est la différence de luminance de 2 couleurs, comprise entre

  • 1:1 (pas de constraste, c'est la même couleur)
  • 21:1 (contraste maximum : noir sur blanc)

Mauvais exemple :

contraste : 1,42:1

Bon exemple :

contraste : 10,36:1

1. Les contrastes

La révolution APCA

Les ratios définis par le WCAG basés sur la luminance sont une bonne base mais les résultats sont parfois déroutants. L'algorithme APCA permet d'obtenir un résultat plus fiable et sera utilisé dans les version futures du RGAA.

2. Les alternatives textuelles

Donner du contexte aux utilisateurs non-voyants

Certaines personnes ne peuvent pas voir vos pages, et doivent donc en déduire le sens par rapport à ce que le lecteur d'écran va vocaliser.

Il est donc très important de proposer une alternative textuelle à tous les éléments graphiques de la page : les images, les videos, les infographies, etc.

3. Les titres

Hierarchisez votre information

La structure de titres joue un rôle important dans la compréhension de votre page web. Les lecteurs d'écran se servent des titres pour créer une table des matières de la page.

Il existe 6 niveaux de titre. Il est important de respecter un ordre cohérent.

```html

Formation A11Y

Introduction

Préambule

Définitions

le cadre légal

Les fondamentaux

Les contrastes

Les alternativrs textuelles

Les titres

La balise title

les balises Hn

```
